# Invoice Payment Tracker

A complete, responsive Flutter application for tracking invoices, logging payments, and viewing financial metrics.

## Features
- **Dashboard**: View total revenue, outstanding amounts, and recent activity.
- **Invoice Management**: Create, edit, and track invoices with different statuses (Pending, Partial, Paid).
- **Payment Logging**: Record payments against specific invoices, supporting partial payments and automatic status updates.
- **Client Management**: Filter and manage invoices by client.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop views.

## Tech Stack
- Flutter
- Provider (State Management)
- Material 3 Design

## Setup
1. Ensure Flutter is installed.
2. Run `flutter pub get`.
3. Run `flutter run` to launch the app on your preferred device or emulator.
